# Conversation Flow

> Guidelines for content organization, user journeys, and information hierarchy.

---

## User Journey Patterns

### TOFU (Top of Funnel) - Awareness
**Goal:** Attract visitors, build awareness

**Content Types:**
- Educational blog posts
- Definition guides
- Industry news
- "What is..." content

**Keywords:** what is, how does, beginner guide, introduction

**Example Paths:**
```
Landing → Blog Post → Related Post → Newsletter Signup
Search → Blog Post → Product Page → Sign Up
```

---

### MOFU (Middle of Funnel) - Consideration
**Goal:** Educate prospects, build trust

**Content Types:**
- Comparison guides
- Feature explanations
- Use cases
- Case studies

**Keywords:** vs, comparison, features, benefits, review

**Example Paths:**
```
Blog Post → Comparison → Free Trial → Demo Request
Product Page → Case Study → Pricing → Sign Up
```

---

### BOFU (Bottom of Funnel) - Decision
**Goal:** Convert prospects to customers

**Content Types:**
- Pricing pages
- Implementation guides
- Guarantees
- Testimonials

**Keywords:** get started, pricing, sign up, free trial

**Example Paths:**
```
Comparison → Free Trial → Paid Plan
Case Study → Demo → Enterprise Plan
```

---

## Content Sequencing

### Blog Post Flow

```markdown
# Title (Include Keyword)

## Introduction
- Hook the reader
- State the problem
- Preview the solution

## Main Content (H2s)
- Point 1
- Point 2
- Point 3

## Conclusion
- Recap key points
- CTA to related content
- CTA to product
```

### Guide Flow

```markdown
# Guide Title

## Prerequisites
- What reader needs
- Tools required

## Step 1: [Action]
- Detailed instructions
- Screenshots/code

## Step 2: [Action]
- Detailed instructions

## Common Issues
- Troubleshooting tips

## Next Steps
- Advanced topics
- Related guides
```

## Internal Linking Strategy

### Link Placement

| Position | Purpose |
|----------|---------|
| Introduction | Link to foundational content |
| Body | Link to related topics |
| Conclusion | Link to CTAs, products |
| Sidebar | Link to category index |

### Anchor Text Rules

- Use descriptive anchor text
- Include keyword when natural
- Avoid "click here"
- Keep under 50 characters

## Information Hierarchy

### Page Priority

1. **H1** - Main title (only one per page)
2. **H2** - Major sections
3. **H3** - Subsections
4. **H4** - Details (avoid if possible)

### Content Density

- One idea per paragraph
- Max 4 sentences per paragraph
- Use bullet points for lists
- Use tables for comparisons

## CTA Placement

### Blog Post CTAs

| Location | CTA Type |
|----------|----------|
| Top (optional) | Newsletter signup |
| Middle | Related blog post |
| End | Product trial, consultation |

### Guide CTAs

| Location | CTA Type |
|----------|----------|
| End of each step | Next step |
| Conclusion | Free trial, demo |

## Responsive Considerations

- Test on mobile, tablet, desktop
- Keep paragraphs short on mobile
- Use adequate touch targets
- Prioritize content hierarchy
